"Gaeta"  Plugin  is a Cesium viewer of QGIS data
-------------------------------------------------

Images:

To test "Gaeta"  you need to have  a web browser with WebGL.

"Gaeta" plugin starts a localhost python miniWebserver

python -m CGIHTTPServer 8000

Use "Gaeta" plugin to see your data inside a Cesium WebGL 3D globe.
